# Changelog

All notable changes to this project will be documented in this file.

## [1.1.2] - 2024-11-15

### Added
- **Visual Designer Interface** - Complete drag-and-drop template designer
- Designer controller with full API endpoints
- Routes for designer interface and API
- Blade component for displaying generated images grid
- View publishing support
- Live preview generation
- Real-time element positioning
- Text field configuration UI
- Element upload and management
- Integration with Filament resources

### Changed
- Updated ServiceProvider to load views and routes
- Enhanced Filament resource with Designer button
- Improved documentation with visual designer usage

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Dynamic image generation with text overlays
- Template-based image composition
- Multi-language support (Arabic, English)
- Image overlay support with opacity
- Flexible text positioning and alignment
- Custom font and color definitions
- Database models for template management
- Filament admin panel integration
- Storage disk configuration
- HTTP response output
