# Default Fonts

This directory contains default fonts that will be published to your application's `storage/fonts` directory.

## Included Fonts

- **Roboto-Regular.ttf** - Default English font
- **NotoKufiArabic-Regular.ttf** - Default Arabic font

## Adding Your Own Fonts

After publishing, you can add your custom fonts to:
- `storage/app/public/fonts/` (recommended)
- or `public/fonts/`

Then configure them in `config/dynamic-image-composer.php`
